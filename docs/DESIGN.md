> Документ описывает визуальный язык проекта: цвета, шрифты, размеры, компоненты.

## 1. Принципы

Интерфейс рассчитан на партнёров старшего возраста с низкой цифровой грамотностью. Из этого следуют все решения ниже.

- Тёмный фон, светлый текст, фиолетовый акцент
- Крупные элементы, большие touch-зоны (минимум 48×48px для кнопок)
- Высокий контраст (текст не серый-на-сером)
- Минимум элементов на экране — пожилым легче, когда мало кнопок
- Никаких выпадающих меню, hover-эффектов, скрытой навигации — всё на виду
- Никаких всплывающих окон, исчезающих тостов — сообщения видны, пока не закроют

## 2. Цвета

```
Фон основной:     #0F0E13
Фон карточки:     #1A1825
Фон поднятый:     #252238
Текст основной:   #F5F3FF
Текст вторичный:  #A8A3C7
Текст приглушён:  #6E6A8A

Акцент основной:  #8B5CF6  (фиолетовый, для кнопок, ссылок, выделений)
Акцент hover:     #A78BFA
Акцент фон:       #8B5CF61A  (прозрачный фиолетовый для подложек)

Статусы:
Успех:   #10B981
Ошибка:  #EF4444
Внимание:#F59E0B
```

### Применение

- `Фон основной` — `body`, общий фон страницы
- `Фон карточки` — все блоки контента: карточки проектов, формы, навигация
- `Фон поднятый` — hover-состояния, активные элементы, акцентированные блоки
- `Текст основной` — заголовки, основной текст
- `Текст вторичный` — описания, подписи, метаданные
- `Текст приглушён` — placeholder в формах, неактивный текст
- `Акцент основной` — все кнопки действия, активные ссылки, выделение текущего пункта меню
- `Статусы` — бейджи статуса проекта, сообщения об успехе/ошибке

## 3. Шрифт

**Inter** — современный, отлично читается на любых размерах, бесплатный, поддерживает кириллицу.

Подключение в Next.js:

```tsx
import { Inter } from 'next/font/google'

const inter = Inter({ subsets: ['latin', 'cyrillic'] })
```

### Размеры

| Применение | Размер | Толщина | Высота строки |
|---|---|---|---|
| Заголовок страницы (`h1`) | 28px | 700 | 1.2 |
| Заголовок секции (`h2`) | 20px | 600 | 1.2 |
| Текст основной | 18px | 400 | 1.5 |
| Текст вторичный | 16px | 400 | 1.5 |
| Подпись | 14px | 400 | 1.4 |

Текст основной 18px (а не стандартные 16px) — сознательное решение для пожилых пользователей. Глазам комфортнее.

---

## 4. Отступы

Используем шаг 8px: `8, 16, 24, 32, 48, 64`.

| Где | Сколько |
|---|---|
| Внутри кнопки | 12px по вертикали, 24px по горизонтали |
| Внутри поля ввода | 14px по вертикали, 16px по горизонтали |
| Внутри карточки | 24px |
| Между блоками на странице | 32px |
| Между секциями | 48px |
| Поля страницы (мобилка) | 16px |
| Поля страницы (десктоп) | 32px |

---

## 5. Скругления

- Кнопки и инпуты: `12px`
- Карточки: `16px`
- Бейджи статуса: `999px` (полностью круглые края)
- Аватары: круглые (`50%`)

## 6. Компоненты

### Кнопка основная

```
Фон: #8B5CF6
Текст: #F5F3FF, 18px, weight 600
Padding: 12px 24px
Border-radius: 12px
Минимальная высота: 48px

Hover: фон #A78BFA
Active: scale 0.98
Disabled: opacity 0.5, cursor not-allowed
Loading: спиннер вместо текста, не кликается
```

Tailwind-эквивалент:
```html
<button class="bg-violet-500 hover:bg-violet-400 text-violet-50 text-lg font-semibold px-6 py-3 rounded-xl min-h-12 active:scale-[0.98] transition disabled:opacity-50 disabled:cursor-not-allowed">
  Войти
</button>
```

### Кнопка вторичная

```
Фон: прозрачный
Border: 1px solid #252238
Текст: #F5F3FF
Остальное как у основной
Hover: фон #252238
```

### Поле ввода

```
Фон: #1A1825
Border: 1px solid #252238
Текст: #F5F3FF, 18px
Padding: 14px 16px
Border-radius: 12px
Высота: 52px

Focus: border #8B5CF6
Placeholder: #6E6A8A
Error: border #EF4444
```

Над полем — лейбл (14px, цвет `#A8A3C7`).
Под полем при ошибке — текст ошибки (14px, цвет `#EF4444`).

### Карточка проекта

```
Фон: #1A1825
Padding: 24px
Border-radius: 16px

Если кликабельна:
  Hover: фон #252238
  Cursor: pointer

Содержимое:
  - Заголовок проекта (20px, weight 600)
  - Описание (16px, цвет #A8A3C7, не больше 2 строк, обрезается с многоточием)
  - Снизу: бейдж статуса + опционально количество участников
```

### Бейдж статуса

```
Padding: 4px 12px
Border-radius: 999px
Шрифт: 14px, weight 500

Цвета:
  active:   фон #10B98133, текст #10B981  (текст «Активный»)
  paused:   фон #F59E0B33, текст #F59E0B  (текст «На паузе»)
  archived: фон #6E6A8A33, текст #A8A3C7  (текст «В архиве»)
```

### Навигация

На мобильных — фиксированная снизу. На десктопе — слева или сверху.

```
Высота: 64px (мобилка)
Фон: #1A1825 с тенью

Пункт меню:
  Иконка: 24×24, lucide-react
  Подпись: 12px под иконкой
  Активный: иконка и подпись цветом #8B5CF6
  Неактивный: цвет #A8A3C7

Touch-зона: минимум 48×48px на каждый пункт
```

Пунктов в навигации — не больше 5. Иконки **обязательно** с подписями (без подписей пожилые не понимают).

### Модальное окно подтверждения

Для разрушительных действий («Удалить проект?»).

```
Фон затемнения: #0F0E13 с opacity 0.7
Окно:
  Фон: #1A1825
  Border-radius: 16px
  Padding: 32px
  Максимальная ширина: 400px
  По центру экрана

Содержимое:
  - Заголовок (20px, weight 600)
  - Текст (18px, цвет #A8A3C7)
  - Две кнопки в ряд: «Отмена» (вторичная), «Удалить» (фон #EF4444)
```

### Сообщение об ошибке/успехе

**НЕ всплывающее.** Постоянное, остаётся пока пользователь не уйдёт со страницы или не закроет.

```
Padding: 16px 20px
Border-radius: 12px
Текст: 16px

Ошибка:    фон #EF444433, текст #EF4444, иконка слева
Успех:     фон #10B98133, текст #10B981, иконка слева
Внимание:  фон #F59E0B33, текст #F59E0B, иконка слева
```

## 7. Состояния

Каждый интерактивный элемент должен иметь чёткие состояния:

- **Обычное** — как описано в компоненте
- **Hover** — фон чуть светлее (на мобилке не применяется)
- **Active/Pressed** — фон ещё светлее, лёгкое уменьшение (`scale 0.98`)
- **Focus** — фиолетовая обводка (важно для доступности и навигации с клавиатуры)
- **Disabled** — opacity 0.5, курсор `not-allowed`
- **Loading** — спиннер вместо текста, элемент не кликается

---

## 8. Иконки

Библиотека: **lucide-react**.

Размеры:
- В навигации: 24×24
- В кнопках: 20×20 (слева от текста, отступ 8px)
- В заголовках карточек: 20×20
- В крупных пустых состояниях: 48×48

Цвет иконок наследуется от цвета текста родителя (через `currentColor`).

---

## 9. Адаптивность

Mobile-first. Брейкпоинты Tailwind по умолчанию:

- `sm`: 640px — небольшие планшеты
- `md`: 768px — планшеты
- `lg`: 1024px — десктоп
- `xl`: 1280px — широкий десктоп

На мобилке:
- Карточки занимают всю ширину
- Навигация снизу
- Шрифты остаются крупными (не уменьшаем!)
- Поля страницы — 16px

На десктопе:
- Карточки в сетке (2-3 в ряд)
- Навигация слева или сверху
- Поля страницы — 32px
- Максимальная ширина контента — 1200px, центрировано

## 10. Что не делать (anti-patterns)

- ❌ Серый текст на тёмно-сером фоне (читать невозможно)
- ❌ Кнопки меньше 44px по высоте
- ❌ Тонкие шрифты (weight < 400)
- ❌ Гамбургер-меню на десктопе
- ❌ Выпадающие меню для основных действий
- ❌ Больше 5-7 пунктов меню одновременно
- ❌ Иконки без подписей в навигации
- ❌ Анимации длиннее 200ms
- ❌ Автоматически исчезающие сообщения (тосты на 3 секунды)
- ❌ Hover-эффекты как единственный способ что-то увидеть
- ❌ Светлая тема (пока — только тёмная)
- ❌ Тени и градиенты (только плоские поверхности)
